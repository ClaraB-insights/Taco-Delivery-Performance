#Taco Delivery Performance Dashboard (Power BI).
     This Power BI dashboard provides insights into taco delivery performance across multiple restaurants and cities. It focuses on revenue, order trends, taco types and sizes, delivery duration and customer tipping behavior to support business decisions and identify opportunities for growth.

*Key Metrics
 - Total Revenue: 6.91K.
 - Average Tip: 1.81.
 - Order Count: 1000.
 - Delivery Duration (Average): 51 minutes.
 - Top Taco Types: Chicken Taco (218 orders), Fish Taco (211), Veggie Taco (197).

*Visual Components
 - Bar Chart: Revenue by restaurant, taco size, and type.
 - Column Chart: Number of taco orders by Order Id.
                 Taco price by size
 - Gauge: Average delivery time.
 - Pie Chart: Weekend vs. weekday orders.
 - Line Chart: Monthly order trends (Jan-Dec).
 - Treemap: Top-performing cities by order volume.

*Key Insights
 - Urban Tacos generate more revenue while El Taco Loco generates the least.
 - Chicken taco had the most orders while beef taco had the least.
 - A significant percentage (72.5%) of orders occur on weekdays.
 - San Antonio and Chicago are top-performing locations.
 - Large tacos generate higher prices, but regular tacos dominate in volume.
